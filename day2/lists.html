목록을 만들때 사용하는 태그

크게 3가지 목록이 있고, 기본적인 문법과 사용방법 익히기

순서가 없는 목록

Unordered List를 줄여서 ul 이라고 하는 명령어로 표현하고, 내부 항목은 List Item이라는 의미를 가진 li

내가 좋아하는 것(음식)

  • 삼겹살
  • 치킨
  • 김치볶음밥

순서가 있는 목록

Ordered List를 줄여서 ol 이라고 하는 명령어로 표현

올해 가고싶은 여행지(순위)

  1. 일본
  2. 프랑스
  3. 미국

사용자 정의 목록

Definition List를 줄여 dl 이라는 태그로 표현하고, 내부 항목은 용어를 뜻하는 Definition Term, dt 태그 그리고 내용은 Definition Description 즉, dd 태그로 표현한다.

일반상식 백과사전

자동차
내연기관으로 움직이는 이동수단으로 사람과 물자를 옮기는데 사용한다.
비행기
운송수단으로 하늘을 나는 기계장치
전동스쿠터
새롭게 등장한 이동수단, 면허가 없어도 이용할 수 있고 시간당 1000원의 비용이 평균

라면을 맛있게 조리하는 방법

끓는물 550ml를 준비하고, 스프를 넣고 면을 넣은 후 3분을 더 가열한뒤, 면을 이리저리 뒤집어서 꼬들꼬들하게 만든다

  1. 냄비에 물 550ml 끓인다.
  2. 물이 끓으면 스프를 넣고 면을 넣는다.
  3. 면이 잘 풀어지면 맛있게 먹는다.
